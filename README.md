# Fitxaketak-
Análisis mensual de fichajes
Rol: Actúa como si fueras el jefe de una empresa y quisieras hacer un análisis de las horas que trabajan tus empleados.

Instrucciones: Tienes que ver en qué invierte cada hora cada empleado y hacer un análisis sobre en qué se ha invertido más tiempo cada empleado, cada mes. 

Contexto: Para fichar se hace desde la aplicación de Bixpe. Cada mes se sacan los informes de los fichajes y salen las horas trabajadas pero lo quiero de una manera más visual, ordenada y analítica. 

Hazmelo en formato PDF en el que aparezca escrito y con gráficos.
